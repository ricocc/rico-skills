# Vitality Style

Vibrant, Creative, Unique, Modern, Dynamic

---

## Design Philosophy

### Brand Philosophy

**"Creativity Unleashed"**

Vitality design philosophy is to inspire creativity and inspiration. Design conveys creative, passionate, and unique brand image through bold vibrant orange and colorful palette, making design work full of vitality and fun.

### Design Philosophy

1. **Bold Colors - Vibrant Energy**
   - High saturation vibrant orange (#FF6B35)
   - Colorful accent system
   - Strong contrast, high visual impact

2. **Gradient Aesthetics - Modern Feel**
   - Bold gradient backgrounds
   - Vibrant orange to passionate pink
   - Creative purple to vibrant orange

3. **Dynamic Design - Full of Energy**
   - Hover zoom, rotation
   - Smooth transition animations
   - Eye-catching

4. **Unique Personality - Non-Conforming**
   - Distinguishes from traditional design tools
   - Bold, avant-garde, experimental
   - For creative people

### Core Keywords

- Vibrant
- Creative
- Unique
- Modern
- Dynamic

## Interaction Philosophy

### Animation Principles

**Dynamic, Attracting, Smooth**

#### Button Feedback

- Hover: Scale (scale 1.05) + rotation (2deg)
- Click: Press effect (scale 0.95)
- Important buttons: Bounce animation

#### Card Interaction

- Hover: Lift 8px + enhanced shadow
- Image: Slight zoom (scale 1.03)
- Icon: Rotate 180deg

#### Transition Duration

- Fast: 100ms (button hover)
- Normal: 200ms (card hover)
- Slow: 300ms (page transition)

**Why Emphasize Dynamics?**
- Attracts attention
- Adds fun
- Inspires creative thinking
- Fits creative agency positioning

### Feedback Mechanisms

**Instant, Visual**

- Operation success: Green + checkmark animation
- Operation failure: Red + shake
- Loading state: Colorful spinner
- Celebration: Confetti effect

## Layout Principles

### Spacing Strategy

**Generous but Rhythmic**

**Standard Spacing:**

- Minimum: 8px
- Small spacing: 12px, 16px
- Medium spacing: 24px, 32px
- Large spacing: 48px, 64px

**Why Generous?**
- Creativity needs breathing room
- Showcasing work needs whitespace
- Reduces visual pressure

### Border Radius System

**Medium to Large Border Radius**

- Buttons: 12px, 16px
- Cards: 16px, 20px
- Inputs: 8px
- Small elements: 4px

**Why Large Border Radius?**
- Friendly and approachable
- Modern feel
- Distinguishes from traditional design tools
- Fits creative agency positioning

### Content Max Width

- Portfolio display: 1200px
- Blog posts: 800px
- Homepage: Full width

## Common Mistakes

### Common Errors

#### Error 1: Using Low Saturation Colors

- Wrong: Soft orange, gray tones
- Correct: High saturation vibrant orange (#FF6B35)
- Reason: Loses vitality and creativity

#### Error 2: Missing Gradients

- Wrong: Solid backgrounds, solid buttons
- Correct: Bold gradients
- Reason: Loses modern feel and visual impact

#### Error 3: Insufficient Animations

- Wrong: Simple transitions, no feedback
- Correct: Zoom, rotation, bounce
- Reason: Loses dynamism and attraction

#### Error 4: Border Radius Too Small

- Wrong: 4px, 6px border-radius
- Correct: 12px, 16px large border-radius
- Reason: Loses modern feel and friendliness

#### Error 5: Too Conservative

- Wrong: Traditional design, conservative layout
- Correct: Bold, experimental, avant-garde
- Reason: Doesn't fit creative agency positioning

#### Error 6: Missing Colorful Elements

- Wrong: Monochromatic, missing accent colors
- Correct: Purple, pink, yellow and other colorful palette
- Reason: Loses vitality and creativity

### Negative Checklist

- Don't use low saturation colors, use high saturation colorful
- Don't remove gradients, use bold gradients
- Don't reduce animations, increase dynamic interactions
- Don't use small border-radius (< 8px), use medium-large (12-16px)
- Don't be conservative, stay bold and avant-garde
- Don't use monochromatic, use colorful palette
- Don't use traditional design, stay modern and experimental

## Animations

```css
--duration-fast: 0.2s;
--duration-normal: 0.3s;
--duration-slow: 0.5s;
--duration-slower: 0.8s;

--ease-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);
--ease-elastic: cubic-bezier(0.68, -0.6, 0.32, 1.6);
```

### Animation Characteristics

- **Elastic Zoom:** Button hover slight zoom (scale 1.02-1.05)
- **Tilt Rotation:** Card hover slight rotation (rotate 1-2deg)
- **Colored Shadows:** Shadow becomes brand color on hover
- **Gradient Flow:** Background gradient animation
- **Staggered Animation:** Elements appear sequentially

```css
/* Dynamic hover */
@keyframes pulse-glow {
  0%, 100% {
    box-shadow: 0 0 20px rgba(255, 107, 53, 0.3);
  }
  50% {
    box-shadow: 0 0 40px rgba(255, 107, 53, 0.5);
  }
}

/* Gradient flow */
@keyframes gradient-shift {
  0% {
    background-position: 0% 50%;
  }
  50% {
    background-position: 100% 50%;
  }
  100% {
    background-position: 0% 50%;
  }
}

/* Elastic entrance */
@keyframes bounce-in {
  0% {
    opacity: 0;
    transform: scale(0.8);
  }
  50% {
    transform: scale(1.05);
  }
  100% {
    opacity: 1;
    transform: scale(1);
  }
}
```
